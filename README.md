# Tiny3D
Tiny 3D demo for Open Appli Player (au mobile phone)

Maybe it can run on other MIDP platforms.

## Operation

- ↑: Look up
- ↓: Look down
- ←: Look left
- →: Look right
- 1: Tilt to the left
- 3: Tilt to the right
- 2: Zoom in
- 8: Zoom out
- 0: Return to the initial position

- *: 1 screen mode
- #: 2 screens mode
